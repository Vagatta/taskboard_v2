## Descripción

`taskboard` es una aplicación web de tipo “tablero Kanban / gestor de tareas”, pensada para organizar tareas o proyectos de forma visual, mediante listas/columnas y tarjetas/tareas.  
El objetivo es ofrecer una versión ligera, sencilla y personalizable, aprovechando tus conocimientos de frontend, sin depender de soluciones externas.  

---

## Características

- Crear listas/columnas.  
- Crear tareas dentro de listas, con título y descripción.  
- Posibilidad de mover tareas entre listas (drag & drop).  
- Interfaz responsiva (móvil / escritorio).  
- Uso de CSS con Tailwind CSS.  
- Estructura modular: frontend en `src/`, assets en `public/`, etc.  
---

## Tecnologías

Este proyecto está construido usando:

- JavaScript / TypeScript (según lo que uses)  
- Tailwind CSS (`tailwind.config.js`)  
- Bundler / gestor de paquetes (npm / yarn — definido en `package.json`)
